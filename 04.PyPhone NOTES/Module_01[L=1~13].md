# 📘 Module 1 Cheat Sheet – Python Fundamentals
- **Variable:** `name = value` (str/int/float)
- **Type check:** `type(x)`
- **Input:** `input("prompt")` → always str
- **Cast:** `int()`, `float()`, `str()`
- **Escape:** `\n`, `\t`, `\\`, `\"`, `\'`
- **Arith:** `//` floor, `%` modulus, `**` power
- **Compare:** `==`, `!=`, `>`, `<`, `>=`, `<=`
- **Logic:** `and`, `or`, `not`
- **Assign:** `+=`, `-=`, `*=`, `/=`, `//=`, `%=`, `**=`
- **Bitwise:** `&`, `|`, `^`, `~`, `<<`, `>>`
