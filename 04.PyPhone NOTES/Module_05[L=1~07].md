# 📘 Module 5 Cheat Sheet – Dictionaries, Tuples, Sets
- **Dict:** `{key: val}` – mutable, keys must be immutable
- **Access:** `d[key]`, `d.get(key, default)`
- **Add/Update:** `d[key] = val`, `d.update({...})`
- **Remove:** `del d[key]`, `d.pop(key)`
- **Methods:** `.keys()`, `.values()`, `.items()`
- **Tuple:** `(a, b)` – immutable
- **Set:** `{a, b}` – unordered, unique
- **Set ops:** `|` union, `&` intersect, `-` diff, `^` sym diff
