# 📘 Module 6 Cheat Sheet – File Handling & Exceptions
- **Read:** `with open('f','r') as f: content = f.read()`
- **Write:** `with open('f','w') as f: f.write(txt)`
- **Append:** `with open('f','a') as f: f.write(txt)`
- **JSON:** `json.dump(obj, f)`, `json.load(f)`
- **try/except:** `try: ... except ErrType: ...`
- **else:** runs if no error
- **finally:** always runs
