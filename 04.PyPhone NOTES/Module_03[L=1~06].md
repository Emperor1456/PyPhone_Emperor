# 📘 Module 3 Cheat Sheet – Strings & Lists
- **Index:** `s[i]`, `s[-1]`
- **Slice:** `s[start:stop:step]`
- **Reverse:** `s[::-1]`
- **Methods:** `.upper()`, `.lower()`, `.split()`, `.join()`, `.replace()`
- **List:** `[a, b]` – mutable
- **List methods:** `.append()`, `.insert()`, `.remove()`, `.pop()`, `.sort()`
- **List comp:** `[expr for x in list if cond]`
- **Iterate:** `for item in list:`
