# 📘 Module 8 Cheat Sheet – Advanced Python
- **Comprehensions:** `[expr for x in iter if cond]`
- **Dict comp:** `{k:v for ...}`
- **Generator:** `(expr for x in iter)` – lazy
- **yield:** makes function a generator
- **Decorator:** `def dec(func): def wrapper(): ... return wrapper`
- **Error best:** catch specific, keep try small, use else/finally
- **CSV:** `csv.reader`, `csv.DictReader`, `csv.writer`
- **venv:** `python -m venv .venv` → `source .venv/bin/activate`
- **Testing:** `assert cond`, `unittest.TestCase`
