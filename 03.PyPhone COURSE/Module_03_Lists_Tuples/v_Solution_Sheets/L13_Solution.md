# L13 Solution – List Creation, Indexing & Mutability

## 🟢 Easy 1 – Create Inventory List
```python
products = ['Laptop', 'Mouse', 'Keyboard']
print(products)
```

## 🟢 Easy 2 – Access by Index
```python
nums = [10, 20, 30, 40]
print(nums[2])
```

## 🟡 Medium 1 – Update Stock
```python
stock = [150, 89, 320, 45]
stock[1] = 95
print(stock)
```

## 🟡 Medium 2 – Length of a List
```python
weights = [12, 8, 15, 10, 7]
print(len(weights))
```

## 🔴 Hard 1 – Swap Two Products
```python
items = ['A', 'B', 'C', 'D']
items[0], items[1] = items[1], items[0]
print(items)
```

## 🔴 Hard 2 – Nested List Access
```python
matrix = [[1,2],[3,4],[5,6]]
print(matrix[1][0])
```
