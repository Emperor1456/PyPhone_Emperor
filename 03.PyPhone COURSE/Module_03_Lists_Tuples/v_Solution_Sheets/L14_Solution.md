# L14 Solution – List Methods – Add, Remove, Sort

## 🟢 Easy 1 – Append to List
```python
items = [1, 2, 3]
items.append(4)
print(items)
```

## 🟢 Easy 2 – Sort a List
```python
lst = [3, 1, 5]
lst.sort()
print(lst)
```

## 🟡 Medium 1 – Insert at Specific Position
```python
tasks = ['write code', 'test', 'deploy']
tasks.insert(1, 'review')
print(tasks)
```

## 🟡 Medium 2 – Remove by Value
```python
items = ['a', 'b', 'c', 'b']
items.remove('b')
print(items)
```

## 🔴 Hard 1 – Pop and Print
```python
lst = [1, 2, 3]
popped = lst.pop()
print(popped)
print(lst)
```

## 🔴 Hard 2 – Reverse Order
```python
lst = [1, 2, 3]
lst.reverse()
print(lst)
```
