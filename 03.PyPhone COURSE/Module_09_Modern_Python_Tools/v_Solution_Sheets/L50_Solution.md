# L50 Solution – Virtual Environments & pip freeze

## 🟢 Easy 1 – Create venv Command
```python
print('python -m venv myenv')
```

## 🟢 Easy 2 – Activate venv Command
```python
print('source myenv/bin/activate')
```

## 🟡 Medium 1 – Freeze Requirements
```python
print('pip freeze > requirements.txt')
```

## 🟡 Medium 2 – Install from Requirements
```python
print('pip install -r requirements.txt')
```

## 🔴 Hard 1 – Deactivate Command
```python
print('deactivate')
```

## 🔴 Hard 2 – Full Workflow
```python
print('python -m venv myenv')
print('source myenv/bin/activate')
print('pip freeze > requirements.txt')
```
