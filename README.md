# 👑 Emperor's Python World

**A mobile‑first, zero‑to‑hero Python curriculum — entirely free and open‑source.**

This repository is the complete journey of one future computer science engineer who started from absolute zero on an Android phone.  
Every line of code was typed by hand, every concept understood, every lesson committed to memory.  
Now, it’s yours.

---

## 📱 Why this course exists

Laptops are not a privilege everyone has.  
With only an Android phone, a code editor (Acode), and a terminal (Termux), I built this roadmap to learn Python the way it *should* be learned – by thinking, not by copying.

This course assumes nothing.  
If you can read and tap a screen, you can start.

## 🧠 How to use this repository

### 1. Learn with the lesson files
Open any `L-*.py` file in order. They are numbered sequentially.  
Each one introduces exactly **two new concepts** and ends with a tiny practice exercise.  
Type every line yourself – never copy‑paste.

### 2. Activate your personal AI mentor
The file **`Python_Curriculum_[Ai_Prompt].md`** contains a ready‑to‑use prompt.  
Copy its entire content and paste it as the first message in a new conversation with any modern AI assistant (ChatGPT, Claude, etc.).  
The AI will instantly become your personal Python mentor and guide you through the exact same "two bricks at a time" method that built this course.

### 3. Use YouTube and books as reinforcements
- Watch short Python tutorials on YouTube after reading a concept.
- Recommended books:  
  - *Python Crash Course* by Eric Matthes  
  - *Automate the Boring Stuff with Python* by Al Sweigart  
  - *Think Python* by Allen B. Downey

### 4. Follow the Getting Started guide
For detailed setup instructions, read **[GETTING_STARTED.md](GETTING_STARTED.md)**.
