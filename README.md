# 👑 PyPhone – Python from a Phone
**A free, mobile‑first Python curriculum. No laptop needed.**

Built entirely on an Android phone using Termux and Acode.  
This course takes you from absolute zero to a full‑stack Python engineer – one typed line at a time.  
Now open‑source for everyone.

---

## 📂 What’s inside

Think of this repo as a complete learning kit. Every folder has a clear job:

**01.PyPhone PLUGIN/** – Your starting point.  
Setup guides that turn an Android phone into a full coding machine (Termux, Acode, Git, GitHub).

**02.PyPhone SYLLABUS/** – The big picture.  
The complete course outline plus the AI prompt that becomes your personal Python mentor.

**03.PyPhone COURSE/** – The core lessons.  
Every topic is a separate `.py` file, numbered in order. Start at `L-01` and work forward.

**04.PyPhone NOTES/** – Quick revision.  
One‑page cheat sheets for each module, designed to fit your phone screen perfectly.

**05.PyPhone Practise/** – Prove your skills.  
Hands‑on practice sets that mix everything from the module into real challenges.

---

**What you’ll learn, in order:**  
Variables → User Input → Operators → Control Flow → Strings → Functions → Data Structures → OOP → Advanced Python → File Handling → APIs → Flask Web Apps → AI Projects → Git & GitHub → Career Readiness.

For the full topic‑by‑topic checklist, open `02.PyPhone SYLLABUS/PyPhone_Curriculum.md`.

   ---

## 🚀 How to start

1. Read the `SETUP_GUIDE.md` inside `01.PyPhone PLUGIN/`. It installs Termux, Acode, Git, and links your phone to GitHub.
2. Copy the AI prompt from `02.PyPhone SYLLABUS/PyPhone_Curriculum[Ai_Prompt].md` into your AI assistant (DeepSeek, ChatGPT, etc.). It becomes your personal mentor.
3. Open `L-01` inside `03.PyPhone COURSE/PyPhone_Module_01/` and start typing.

No copy‑paste – you write every line. That’s how real learning happens.

---

## 🧠 The method

- **Two bricks at a time** – exactly two new concepts per lesson, never overwhelming.
- **Phone‑first design** – all code fits a phone screen, no sideways scrolling.
- **AI‑assisted, not AI‑dependent** – you understand every single line you write.
